# Multi-threading-Client_server
This repository contains a robust multi-threaded Client-Server architecture developed for network security coursework. The system facilitates secure communication between clients and a server using TCP sockets.

Key Features:

Multi-Threading: Uses POSIX threads (pthread) to handle multiple concurrent client connections efficiently.

Custom Encryption: Implements a layered encryption/decryption protocol to ensure data privacy during transmission.

Role-Based Access Control (RBAC): Enforces a strict three-tier privilege system (Entry, Medium, Top), validating both user credentials and command authorization before execution.
